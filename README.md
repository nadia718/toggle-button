# toggle-button

A simple and interactive toggle button project built using HTML, CSS, and JavaScript. The button allows users to switch between different states with smooth visual effects.

## Features

* Toggle ON/OFF functionality
* Smooth transition animations
* Interactive user interface
* Beginner-friendly project
* Pure HTML, CSS, and JavaScript implementation

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

toggle-button/
│
├── index.html
├── style.css
└── script.js

## Project Goal

This project was created to practice JavaScript event handling, DOM manipulation, and CSS transitions while building an interactive toggle switch.

## Functionality

* Switch between ON and OFF states
* Dynamic UI updates
* Smooth animation effects
* User-friendly interaction
